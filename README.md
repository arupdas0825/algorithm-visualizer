## 🚀 About Algomate

**Algomate** is not just another algorithm visualizer; it is a **Neural Interface for Data Structures and Algorithms**. Built with the philosophy that learning complex logic should be an immersive, cinematic experience, Algomate transforms abstract DSA concepts into vibrant, tactile visualizations.

### Why Algomate?
Traditional visualization tools often feel static and academic. Algomate bridges the gap between **education** and **interaction** by utilizing:
- **Cinematic Motion**: High-tension spring physics and fluid transitions.
* **Neural Aesthetics**: A premium UI inspired by modern AI operating systems.
* **Interactive Discovery**: Real-time manipulation of data structures with instant visual feedback.

---

## ✨ Core Features

### 📊 Visualization Engines
- **Sorting Visualizer**: Real-time execution of Bubble, Merge, and Quick sort with variable clock speeds and comparison tracking.
- **Graph & Pathfinding**: Interactive grid system for BFS, DFS, and Dijkstra's algorithm with draggable endpoints and walls.
- **Data Structures Hub**: Visual exploration of Stacks, Queues, Linked Lists, and Trees with interactive push/pop/insert operations.
- **Dynamic Programming**: Step-by-step state transition visualization for classic DP problems.

### 🧠 Intelligent Systems
- **AI Tutor Beta**: Context-aware complexity analysis and algorithmic recommendations.
- **Comparison Mode**: Side-by-side performance benchmarking of multiple algorithms.
- **Learning Hub**: Integrated documentation and logic breakdowns for every visualizer.

### 🎨 OS-Level Customization
- **Global Theme Engine**: Swap between Dark, AMOLED, Cyberpunk, Neon Glow, and Light Protocol modes instantly.
- **Neural Control Center**: Viewport-aware settings panel for fine-tuning motion physics, sonic environments, and visualizer performance.

## 🛠️ Tech Stack

### Frontend & Core
- **React 18**: Component-based UI architecture.
- **Tailwind CSS**: Utility-first styling with HSL variable-based dynamic theming.
- **Framer Motion**: High-performance spring physics and layout animations.

### State & Logic
- **Zustand**: Lightweight, persistent state management for user profiles and system settings.
- **React Router 6**: Client-side navigation with cinematic route transitions.
- **Lucide React**: Premium icon system.

## 📐 Project Architecture

```text
algomate/
├── src/
│   ├── components/       # UI Components & Modules
│   │   ├── common/       # Reusable Atomic Elements
│   │   ├── layout/       # Sidebar, Navbar, Page Wrappers
│   │   ├── modals/       # Control Center, Profile Edits
│   │   └── visualizers/  # Algorithm-specific logic
│   ├── context/          # Theme & Global Contexts
│   ├── pages/            # Main Route Views
│   ├── store/            # Zustand State Definitions
│   ├── styles/           # Global CSS & Tailwind Tokens
│   └── utils/            # Math & Sound Engine Logic
├── public/               # Static Assets & Logos
└── tailwind.config.js    # Theme Variable Mapping
```

## 🚦 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/arupdas0825/algorithm-visualizer.git
   cd algorithm-visualizer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch the Neural Interface**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```
